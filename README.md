# darknet-yolo
ROS yolo based on [darknet](https://github.com/pjreddie/darknet)

### Running environment
```
ROS melodic
Ubuntu 18.04 (virtual machine) / 16.04 (Nvidia nano jetson)
```

### Default Parameters
```
Darknet
GPU=0
CUDNN=0
OPENCV=0

my_image_detector (ROS launch file)
target = multiball
cam_width = 800
cam_height = 400
zoom_width = 400
zoom_height = 400
crop_frame = true
```
